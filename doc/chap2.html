2 Isomorphism to Generic Wreath Product

2.1 Overview

In GAP the command WreathProduct(K, H) returns a group with a representation that depends on the representation that K is given in. This package provides a method to compute an isomorphism from such wreath products to the generic wreath product provided in this package.

2.1-1 IsomorphismToGenericWreathProduct
‣ IsomorphismToGenericWreathProduct( G )( operation )

returns an isomorphism from a specialized wreath product G to a generic wreath product.

2.2 Permutation representation

If K is given as a permutation group the command WreathProduct(K, H) in GAP returns the imprimitive wreath product that is embedded into the symmetric group.

Here we list the internal commands that are used to translate between the permutation representation and the generic representation.

2.2-1 WPE_GenericPermWreathProduct
‣ WPE_GenericPermWreathProduct( G )( function )

returns a generic wreath product from a perm wreath product.

2.2-2 WPE_ConvertPermToRep
‣ WPE_ConvertPermToRep( G, W, g )( function )

returns the generic representation of an element g in G, where G is a wreath product in permutation representation and W is the same wreath product in generic representation.

2.2-3 WPE_ConvertRepToPerm
‣ WPE_ConvertRepToPerm( G, W, x )( function )

returns the permutation representation of an element x in W, where G is a wreath product in permutation representation and W is the same wreath product in generic representation.
