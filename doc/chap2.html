2 Functions

Here we include a list of all functions that are provided to the User.

2.1 Function List

The following functions are designed to improve the User experience when working or experimenting with wreath products and their elements. Most functions are about presenting elements in an intuitive way and giving access to useful information.

2.1-1 ComponentsOfWreathProduct
‣ ComponentsOfWreathProduct( W )( function )

returns the groups [K, H] of W = K wr H. The argument W must be a wreath product. (see 1.2)

2.1-2 TopGroupOfWreathProduct
‣ TopGroupOfWreathProduct( W )( function )

returns the top group of the wreath product W. (see 1.2)

2.1-3 BaseGroupOfWreathProduct
‣ BaseGroupOfWreathProduct( W[, i] )( function )

returns the base group of the wreath product W. If the optional argument i is provided, the function returns the i-th factor of the base group of W. (see 1.2)

2.1-4 TopComponentOfWreathProductElement
‣ TopComponentOfWreathProductElement( x )( function )

returns the top component of x. The argument x must be a wreath product element. (see 1.2)

2.1-5 BaseComponentOfWreathProductElement
‣ BaseComponentOfWreathProductElement( x[, i] )( function )

returns the base component of x. If the optional argument i is provided, the function returns the i-th base component of x. The argument x must be a wreath product element and the optional argument i must be an integer. (see 1.2)

2.1-6 Territory
‣ Territory( x )( attribute )

returns the territory of x. The argument x must be a wreath product element. (see 1.3)

2.1-7 IsWreathCycle
‣ IsWreathCycle( x )( attribute )

tests whether x is a wreath cycle. The argument x must be a wreath product element. (see 1.3)

2.1-8 IsSparseWreathCycle
‣ IsSparseWreathCycle( x )( attribute )

tests whether x is a sparse wreath cycle. The argument x must be a wreath product element. (see 1.4)

2.1-9 WreathCycleDecomposition
‣ WreathCycleDecomposition( x )( attribute )

returns the wreath cycle decomposition of x. The argument x must be a wreath product element. (see 1.3)

2.1-10 SparseWreathCycleDecomposition
‣ SparseWreathCycleDecomposition( x )( attribute )

returns the sparse wreath cycle decomposition of x. The argument x must be a wreath product element. (see 1.4)

2.1-11 ConjugatorWreathCycleToSparse
‣ ConjugatorWreathCycleToSparse( x )( attribute )

returns a list of wreath product elements c_1, ..., c_l, such that for each wreath cycle w_k in the decompositon of x, the element w_k^{c_k} is the corresponding sparse wreath cycle in the sparse wreath cycle decomposition of x. The argument x must be a wreath product element. (see 1.4)

2.1-12 Yade
‣ Yade( x[, i] )( attribute )

returns the yade of the wreath cycle x evaluated at the smallest territory point. If the optional argument i is provided, the function returns the yade evaluated at the point i. The argument x must be a wreath cycle and the optional argument i must be an integer. (see 1.3)
